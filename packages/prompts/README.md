# @agentsy/prompts

Prompt optimization and agentic pattern management for the @agentsy platform.

## Features

- **Prompt Optimization** — Classical (few-shot, chain-of-thought) and modern (instruction tuning, structured prompting) techniques
- **Token Integration** — Prompt compression and token budget awareness
- **Memory Integration** — Global system prompts storage, session-level prompt management, versioning
- **Agentic Patterns** — ReAct, task decomposition, reasoning loops
- **Safety & Guardrails** — Structured safety measures and content filtering

## Installation

```bash
npm install @agentsy/prompts
```
