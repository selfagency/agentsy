# `@agentsy/observability`

Telemetry, logging, and monitoring infrastructure.

## Purpose

`@agentsy/observability` provides structured logging, distributed tracing hooks, and metrics emission for Agentsy agents and pipelines, with adapters for common observability backends.

## Status

- Published package.
- See the [roadmap](../roadmap.md) for planned capabilities.

## Notes

This package is currently in active development. API surface is subject to change.
