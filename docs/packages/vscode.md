# @agentsy/vscode

`@agentsy/vscode` is the currently published package in the Agentsy framework.

## Purpose

Provides reusable building blocks for VS Code Language Model Chat Provider extensions:

- provider scaffolding
- stream rendering helpers
- API key management
- settings and usage tracking
- MCP integration helpers

## Install

```bash
npm install @agentsy/vscode vscode
```
