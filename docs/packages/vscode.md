# `@agentsy/vscode`

- **Status:** Published
- **Role:** VS Code chat-provider integration, rendering, settings, usage tracking, and MCP helpers

## Where it fits

`@agentsy/vscode` packages lower-level Agentsy processing patterns into a VS Code-focused surface for Language Model Chat Provider scenarios.

## Boundary rule

This package may expose machine-readable diagnostics and setup hints for VS Code integration, but framework-wide setup/doctor UX belongs to `@agentsy/cli`.
