# `@agentsy/prompts`

Prompt templates and prompt orchestration utilities.

## Purpose

`@agentsy/prompts` provides a typed template system for constructing, composing, and versioning prompts, along with helpers for prompt chaining and conditional message assembly.

## Status

- Published package.
- See the [roadmap](../roadmap.md) for planned capabilities.

## Notes

This package is currently in active development. API surface is subject to change.
