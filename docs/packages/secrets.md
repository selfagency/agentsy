# `@agentsy/secrets`

Secret management and credential handling.

## Purpose

`@agentsy/secrets` provides a unified interface for loading, rotating, and scoping secrets (API keys, tokens, credentials) used by providers and tools, with support for environment variables and external secret stores.

## Status

- Published package.
- See the [roadmap](../roadmap.md) for planned capabilities.

## Notes

This package is currently in active development. API surface is subject to change.
