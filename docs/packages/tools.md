# `@agentsy/tools`

Tool execution framework and common tool implementations.

## Purpose

`@agentsy/tools` provides the tool registration, dispatch, and result-handling infrastructure used by the Agentsy agent loop, along with a set of built-in tool implementations.

## Status

- Published package.
- See the [roadmap](../roadmap.md) for planned capabilities.

## Notes

This package is currently in active development. API surface is subject to change.
