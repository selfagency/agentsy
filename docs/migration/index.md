# Migration Guide

This directory contains migration documentation for upgrading Agentsy packages.

- [@selfagency/llm-stream-parser to @agentsy/\* Migration](./llm-stream-parser.md): Guidance for migrating from `@selfagency/llm-stream-pareser`, from which Agentsy descends.
- [@agentsy/vscode v0.1.x to v0.2.02 Migration](./v0.1-to-v0.2.md): Detailed steps and API changes for upgrading from version 0.1.x to 0.2.x.

Refer to individual package migration guides for in-depth information on API changes and new features.
